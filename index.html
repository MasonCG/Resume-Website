Projects

Unit Test Clone

I built a custom unit testing framework in Python to explore how professional testing tools are designed internally. Rather than relying on Python’s built-in unittest, I implemented core testing concepts from scratch, including test discovery, execution flow, lifecycle management, and result aggregation.

The framework includes optional fail-fast execution, a command-line interface, and structured test reporting with support for JUnit XML output for CI compatibility. The project emphasizes clean architecture and extensibility, mirroring how real world developer tools are structured.

This project strengthened my understanding of testing internals, error handling, CLI design, and building maintainable tooling skills directly applicable to large scale software systems.
